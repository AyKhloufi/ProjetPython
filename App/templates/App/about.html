À propos de Mon Site Recettes

Une application Django moderne pour la gestion de recettes culinaires

Projet Académique

Développé dans le cadre du cours Programmation web et sécurité (420-E60-CH) au Cégep de l'Outaouais.

  • Travail 2 - Prototype Django transactionnel
  • Architecture MVT complète
  • Opérations CRUD avancées

Technologies Utilisées

Application construite avec les technologies web modernes et les meilleures pratiques.

  • Django 5.2.6
  • Python
  • Bootstrap 5.2.3
  • SQLite

Fonctionnalités Principales

Gestion des Recettes

Créez, modifiez et organisez vos recettes préférées avec facilité.

Base d'Ingrédients

Gérez une base complète d'ingrédients réutilisables pour toutes vos recettes.

Unités de Mesure

Système flexible d'unités pour des mesures précises et cohérentes.

Design Responsive

Interface adaptative qui fonctionne parfaitement sur tous les appareils.
